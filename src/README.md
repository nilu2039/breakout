# 🧱 Breakout (1979) — Zig + Raylib

This is a small Breakout-style game inspired by the original **Breakout (1979)**.
I built it using Zig and Raylib while learning how game loops, rendering, and collision systems work.

---

## 🎮 Features

* Paddle movement (left/right)
* Ball physics and collision detection
* Brick breaking system
* Particle effects when bricks are destroyed
* Simple update/render game loop

---

## 🛠 Tech

* Zig
* Raylib

---

## 🚀 Run

Make sure Zig is installed.

```bash 
git clone https://github.com/nilu2039/breakout 
cd breakout
zig build run
```

---

## 🎯 Controls

* ← / → : Move paddle
* ESC : Quit

---

## 💥 Particle Effects

When a brick is destroyed, particles are spawned with random directions and speeds to create a small burst effect.
This makes the game feel a bit more responsive compared to instantly removing bricks.
